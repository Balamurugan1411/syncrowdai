# 🏟️ SynCrowd AI: Next-Gen "God-Mode" Stadium Orchestration


> **SynCrowd AI** is a market-leading cognitive orchestration engine designed for massive stadium environments. Developed for **Google PromptWars Virtual**, it evolves beyond reactive monitoring into a proactive, interactive "God-Mode" command center, surpassing industry benchmarks in predictive intelligence and operational control.

---

## 🚀 Key "God-Mode" Features

### 🧠 Syn AI: Cognitive Assistant
A holographic AI companion that provides real-time "Cognitive Orchestration" insights. Syn moves the platform from a data display to an intelligent partner, predicting anomalies before they manifest.

### ⏳ Temporal "Time-Travel" Engine
Shift the digital twin into the future. Using the **Predictive Horizon** slider, operators can visualize simulated stadium states (T+15, T+30, T+60m) to preemptively mitigate bottlenecks.

### 🕹️ Shadow Control: Interactive 3D Map
Total infrastructure sovereignty. Directly interact with gates, turnstiles, and elevators on the 3D map to manually override status. The simulation engine instantly recalculates crowd flows based on your commands.

### 📢 Geofenced Micro-Casting
Precision communication. Dispatch sector-specific notifications (Staff, VIP, Security) with priority coding, ensuring the right message reaches the right zone at the right time.

### ♿ Accessibility Intelligence
Coordinator for inclusive routing. Features an **AI Inclusion Audit** that proactively identifies comfort threshold violations at critical mobility nodes.

## 🛠️ Technology Stack

- **Core**: React 19 + Vite 8
- **Intelligence**: Custom Simulation & Routing Engines (JS-based)
- **Visuals**: Framer Motion (Micro-animations) + Lucide (Iconography)
- **Styling**: Tailwind CSS 4 + Glassmorphic Design System
- **Data**: Edge Perception Modeling

## 📦 Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Balamurugan1411/syncrowdai.git
   cd syncrowdai/syncrowd-ui
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch the Orchestration Engine**:
   ```bash
   npm run dev
   ```
